h3(#list-of-features). List of features

* Customize feed's body input

h3(#requirements). Requirements

* Textpattern 4.x.x

h3(#installation-and-usage). Usage

p. The plugin uses form called @rah_feed_body@ customize feeds output. The automatic setup should have created it, but if it didn't then create the form, called @rah_feed_body@. This form will hold your rss- and atom-feed contents, or so said the feeds body content. You can use txp-tags as the form will work like any other form.

h3(#example). Example

p. rah_feed_body-form's content:

bc. <txp:excerpt />
<p><txp:permlink>Read more...</txp:permlink></p>

p. The code outputs excerpt and a read more-link.
